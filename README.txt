Folder 3D_rendering. It contains the algorithm for visualizing the metastases in 3D with the axial, sagittal and coronal slices as reference of the patient anatomy.

Please read the main MATLAB function named: plot_brain_metastases.m. In this function are indicated the input required for 3D visualization.
distinguishable_colours.m is an auxiliary function used by plot_brain_metastases.m to plot metastases in 3D with maximal perceptually distinct colours for propely visualizing
the spatially distinct metastases in a patient (no modifications are required in this function)
